# Smart Parking Gate System

## Overview
This project is a Smart Parking Gate System using ATmega16 microcontroller, IR sensors, servo motor, and LCD display.

## Components Used
- ATmega16
- IR Sensors
- Servo Motor
- LCD Display
- LM7805 Voltage Regulator
- Crystal Oscillator

## Features
- Automatic vehicle detection
- Gate opening and closing
- Parking slot monitoring
- LCD status display

## Software Used
- Arduino IDE
- EasyEDA

## Working
When a vehicle is detected by the IR sensor, the servo motor opens the gate automatically. LCD displays parking information and slot availability.
